# fab-inspector-cli

CLI invocation skill for the Fab Inspector command-line tool.

## What's in this skill

The [SKILL.md](SKILL.md) file provides AI agents with complete guidance on:

- **Invocation syntax** — how to call `fab-inspector` with required and optional parameters
- **Parameter reference** — every CLI argument with types, defaults, and descriptions
- **Authentication methods** — six auth modes (local, interactive, clientsecret, certificate, federatedtoken, managedidentity) with required parameters for each
- **Output formats** — CONSOLE, HTML, JSON, PNG, ADO, GitHub and their behaviors
- **Validation constraints** — which parameter combinations are valid or invalid
- **Usage examples** — runnable commands for local, CI/CD, workspace, and hybrid scenarios
- **Sensitive data handling** — security guidance for credentials and tokens

## When to load this skill

Load this skill when a user needs to:

- Run the Fab Inspector CLI
- Set up a CI/CD pipeline with Fab Inspector
- Configure authentication for Fabric workspace access
- Troubleshoot CLI parameter issues
- Understand available output formats
